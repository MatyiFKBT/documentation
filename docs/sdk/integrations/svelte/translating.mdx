There are two ways how to translate with Tolgee.

## `getTranslate` method

Other option how to translate is using of [`getTranslate`](./api#function-gettranslate) function. You can use it this way:

```svelte
<script>
  import { getTranslate } from "@tolgee/svelte";

  // t is store
  const { t } = getTranslate();
</script>

<!-- use the translation function -->
{$t('key', 'Default value')}
```

> Read more about [`tolgee.t`](/docs/sdk/api/core_package/tolgee#t)

### Using namespaces

[`getTranslate`](./api#function-gettranslate) function can be also used for loading additional namespaces.

```ts
const { t, isLoading } = getTranslate('common');
```

This example will add `common` to [active namespaces](/docs/sdk/api/core_package/tolgee#addactivens). You can then also use `isLoading` store, which will tell you if your data are ready.

## The `T` component

Another way is to use [`T` component](./api#t-component)

```svelte
<script>
  import { T } from "@tolgee/svelte";
</script>

<T keyName="this_is_a_key" />
```

## Message format

By default Tolgee is without any message formatter. You can use [`FormatSimple`](/docs/sdk/api/core_package/format-simple) or [`FormatIcu`](/docs/icu-message-format)
